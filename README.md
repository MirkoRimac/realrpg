# RealRPG

A self-hosted, family-friendly RPG-style life organizer.
Turn tasks into quests, earn XP & gold, and grow together as a party.

## Tech Stack
- Next.js (App Router) + TypeScript
- PostgreSQL (Docker)
- Prisma ORM

## Requirements
- Node.js (LTS)
- Docker Desktop
